# TerrariaProj
A simple terraria clone made in C++ using OpenGL.



Use A and D to move.

Hold shift to speed up. 

Space to fly.

F to disable gravity.

0-9 select inventory slot.

LMB to place, RMB to break.

E prints out the inventory content.
